About


FairFare is an Uber vs. Lyft price comparing tool. FairFare utilizes APIs from Uber and Lyft to provide real-time pricing on fares including surge rates. Google Maps' autocomplete feature provides suggestions on destinations for fareseekers. FairFare was built with a mobile-first design.

Try FairFare here:    fairf.herokuapp.com

Languages/Skills Used:

Ruby Rails AJAX JavaScript Uber API Lyft API Google Maps API UJS heroku Bootstrap JSON CURL Commands

About


Window of Time is a Web Game developed with the JavaScript library Phaser. Inspired from the overhead style of the original GTA games, you control a character who has the ability to slow time when they stop moving and defenstrates their foes. All sprites and sprite animations, required extensive use of Photoshop and JSON.

Play Window of Time:    defenestrate.herokuapp.com
